01

Project Thalassa

Autonomous Marine Waste Collection Buoy System

A fully autonomous buoy system designed to detect, collect, and segregate marine waste using computer vision and embedded control. Built on a Raspberry Pi 5 + Arduino Mega dual-board architecture, with a YOLOv8 vision pipeline, LiFePO₄ battery system, brushless thrusters, and a dual conveyor system with a servo-actuated waste segregation gate. Formal thesis documentation covers system architecture, data analysis procedures, financial planning, and a complete evaluation procedure.

Desbank

Question Bank & Exam Generation System

A relational database system that manages an academic question bank with Bloom's Taxonomy classification and automated exam generation. Features a two-level Table of Specifications schema (tos_topics + tos_topic_bloom) in MySQL, supporting weight percentage snapshots and manual override of item counts. Designed for real-world academic deployment with an extensible schema architecture.
